Weave

A fine-grained reactive UI framework — signal-native, tiny, and TypeScript-first.

The screen is a fabric, and reactivity is the thread. Change a value and Weave touches only the one spot that depends on it — no re-renders, no Virtual DOM, no dependency arrays.

Signals all the way down

One reactivity model powers everything from a text node to the router. Nothing else to learn.

No Virtual DOM

Templates compile straight to real DOM updates bound to your signals. The runtime stays genuinely small.

Batteries included

Routing, state, forms, i18n, and motion are first-party and share the same reactive core.

A real IDE citizen

First-class VS Code and WebStorm support — template type-checking and jump-to-definition included.
